En esta clase veremos aspectos b√°sicos de optimizaci√≥n. Espec√≠ficamente, veremos c√≥mo obtener m√°ximos y m√≠nimos de una funci√≥n escalar de una variable (como en c√°lculo diferencial).

¬∂

Criterio de la segunda derivada¬∂

Sea $f(x)$ una funci√≥n tal que $f'(c)=0$ y cuya segunda derivada existe en un intervalo abierto que contiene a $c$.

  • Si $f''(c)>0$, entonces $f(c)$ es un m√≠nimo relativo.
  • Si $f''(c)<0$, entonces $f(c)$ es un m√°ximo relativo.
  • Si $f''(c)=0$, entonces el criterio no decide.

Metodolog√≠a¬∂

  1. Definir el criterio de optimizaci√≥n.
    1. Definiendo una variable simb√≥lica o
    2. desde el entorno def.
  2. Derivar la funci√≥n a optimizar.
  3. Encontrar la ra√≠z de la derivada.
  4. Definir qu√© puntos son m√°ximos o m√≠nimos.
    1. Usando el criterio de la segunda derivada o
    2. evaluando num√©ricamente (m√©todo gr√°fico).

Se desea maximizar la iluminaci√≥n $$ \mathrm{max}I=\mathrm{max}k\frac{\sin(\alpha)}{s^2}, $$ donde, usando el teorema de Pit√°goras, se tiene que $s^2=h^2+2^2$, siendo $2$ el radio de la mesa; adem√°s, por definici√≥n de funciones trigonom√©tricas, sabemos que $\sin(\alpha)=\frac{h}{s}$. Soluci√≥n:

Equlibrando las fuerzas horizontales, es f√°cil ver que, $$ f\cos(\theta)=k(W-f\sin(\theta), $$ donde, el lado izquierdo de la igualdad corresponde a la componente horizontal de la fuerza empleada y el lado derecho, corresponde a la fricci√≥n generada.

sumando en ambos lados $kf\sin(\theta)$, resulta $$ f(\cos(\theta)+k\sin(\theta))=kW, $$ despejando $f$ se tiene: $$ f=\frac{kW}{\cos(\theta)+k\sin(\theta)}. $$ Siendo √©ste el criterio de optimizaci√≥n para $-\pi<\theta<\pi$.

def fuerza(theta, k, W):
    return k * W / (np.cos(theta) + k * np.sin(theta))

import numpy as np
from scipy.optimize import minimize

# Definimos la funci√≥n de tiempo t(x)
def tiempo(x):
    # Distancia desde P a Q
    distancia_barca = np.sqrt(9**2 + (15 - x)**2)
    # Tiempo en la barca (4 km/h)
    tiempo_barca = distancia_barca / 4
    # Tiempo caminando (5 km/h)
    tiempo_caminar = x / 5
    # Tiempo total
    return tiempo_barca + tiempo_caminar

# Definimos el rango de x (0 <= x <= 15)
resultado = minimize(tiempo, 7.5, bounds=[(0, 15)])

# Resultado de la optimizaci√≥n
x_optimo = resultado.x[0]
tiempo_minimo = resultado.fun

print(f"El punto √≥ptimo de desembarco es a {x_optimo:.2f} km del punto m√°s cercano.")
print(f"El tiempo m√≠nimo es de {tiempo_minimo:.2f} horas.")
